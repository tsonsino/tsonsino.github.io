# How to get this live on GitHub Pages 

## One-time setup (~11 minutes)

### 1. Create a GitHub account
Go to github.com and sign up if you don't have one.
Use a professional username — `tsonsino` is perfect.

### 2. Create the repository
- Click the + icon in the top right → "New repository"
- Repository name: `tsonsino.github.io` (must be exact)
- Set to Public
- Click "Create repository"

### 3. Upload your files
- Click "uploading an existing file" on the new repo page
- Drag and drop ALL files from this folder:
  - index.html (your resume site homepage)
  - toolkit/ folder (drag the whole folder)
- Click "Commit changes"

### 4. Enable GitHub Pages
- Go to Settings → Pages (left sidebar)
- Source: Deploy from a branch
- Branch: main / (root)
- Click Save

### 5. Your site is live!
Wait 2-3 minutes, then visit: https://tsonsino.github.io

## Updating the site later
To make edits, come back to Claude with your changes.
Get a new index.html file, go to your GitHub repo,
click on the file, click the pencil icon to edit,
paste the new content, and commit.
